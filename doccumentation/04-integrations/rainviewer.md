# Integration: RainViewer

## Purpose

Ingest radar frame metadata for weather overlay controls.

## Credentials

- Public endpoint path does not require API key for baseline use.

## Module

- `integrations/rainviewer/RainViewerPollingService.java`

## Default Feed

- `https://api.rainviewer.com/public/weather-maps.json`

## Override

- `VISION_INTEGRATIONS_RAINVIEWER_FEED_URL`
