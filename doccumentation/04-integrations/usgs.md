# Integration: USGS Earthquakes

## Purpose

Ingest current earthquake events for live seismic visualization.

## Credentials

- No API key required for default public feed.

## Module

- `integrations/usgs/UsgsPollingService.java`

## Default Feed

- `https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/all_hour.geojson`

## Override

- `VISION_INTEGRATIONS_USGS_FEED_URL`
