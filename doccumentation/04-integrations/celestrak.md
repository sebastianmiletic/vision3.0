# Integration: CelesTrak

## Purpose

Ingest active satellite catalog records for orbital layer rendering.

## Credentials

- No key required for default public query path.

## Module

- `integrations/celestrak/CelesTrakPollingService.java`

## Default Feed

- `https://celestrak.org/NORAD/elements/gp.php?GROUP=active&FORMAT=json`

## Override

- `VISION_INTEGRATIONS_CELESTRAK_FEED_URL`
