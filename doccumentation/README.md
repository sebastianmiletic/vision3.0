# Vision v3.0 Doccumentation Index

This folder is the primary knowledge base for Vision v3.0. Every major runtime surface, integration, and operational path is documented here.

## Navigation

- `00-getting-started/`: first-run setup and environment variables.
- `01-architecture/`: system map, data flow, and module boundaries.
- `02-apps/`: detailed per-application reference.
- `03-frontend/`: UI shell, Cesium globe, and frontend data flow.
- `04-integrations/`: source-by-source integration behavior and requirements.
- `05-api/`: API inventory, endpoint contracts, and request/response examples.
- `06-runbooks/`: local operations, troubleshooting, and testing workflows.
- `07-adr/`: architecture decision records.

## Fast Start

1. Read `00-getting-started/local-development.md`.
2. Configure `.env.local` from `00-getting-started/environment-variables.md`.
3. Start stack with `npm run dev`.
4. Open `http://localhost:8081`.
